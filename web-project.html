Web Project Information

On the homepage (index.html), we provided a brief overview of the team, including its purpose and context, to establish the team's identity and set the tone for the rest of the website. We designed a homepage (index.html) to introduce our team. The page includes a header with the team logo and a navigation bar that connects all sections of the website. The homepage serves as a hub, presenting the team’s identity and offering easy access to the project’s main features. The structure was implemented using HTML5, and CSS was used to style the header, navigation bar, and overall layout. The CSS flexbox model ensured proper alignment and responsiveness.

We then introduced individual profiles for each team member, detailing their background, interests, and roles within the team. The "Team Members" section provides detailed profiles for each member, including names, photos, and links to their Codecademy profiles. The profiles are displayed in individual boxes styled with rounded corners, background colors, and hover effects. We utilized a combination of HTML and CSS to format these boxes and used Flexbox to align them evenly. Each profile includes a functional button with interactive hover effects, styled using custom CSS. These buttons include the Codecademy logo on the left and link to the team members’ respective profiles.

Then we created dedicated section that provides details about our Hour of Code event, including its objectives, activities, and the engagement we facilitated with participants. The page serves as a summary of the event’s key moments and outcomes. Images from the event are included to provide visual context. The layout was styled with CSS to include bordered containers and responsive image scaling for consistency across devices. HTML semantics were used to ensure the content is accessible and structured. This section reflects how the team engaged with participants to promote coding and computational thinking.

We included a section introducing code art created using algorithms, which is spinning spiral with interactive component. The user can pause and continue spinning spiral, and if needed also save the picture when stopped. The "Code Art" section demonstrates a creative application of JavaScript and HTML5 Canvas to generate interactive algorithmic art. The centerpiece is a spinning spiral pattern that users can control using buttons to pause, resume, or save the art. The JavaScript script utilizes randomized parameters for vibrant colors and unique patterns, offering a dynamic and engaging user experience. CSS ensures the canvas is centered on the page, and the buttons are styled to be visually appealing and user-friendly. This section showcases the integration of algorithms and art in a web development context.

In the end, we developed a section about our web project, which highlights the web development project we undertook, describing the pages we developed and individual profiles of team members. The "Web Project" section highlights the web development process and technologies used. This page describes the various tasks completed, such as structuring the team profiles, creating interactive buttons, and designing layouts. For the navigation bar and footer, CSS properties such as position: fixed and flex were used to create a consistent and responsive layout. JavaScript enhanced the functionality by enabling interactivity, such as dynamic content rendering. The CSS variables (--ada-blue and --ada-red) allowed for consistent theming across all pages.
